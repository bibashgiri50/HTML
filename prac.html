HTML

HTML: HTML stands for hyper text mark up language. It is used to design web pages using markup language. HTML is the combination of Hypertext and Markup language. Hypertext defines the line between the web pages. Markup language is used to define the text document within tag which defines the structure of web pages. the language is used to annotate(make notes for the computer) text so thet a machine can understand it and manipulate text accordingly. Most markup languages (e.g HTML)are human readable. Languages uses tags to define what manipulation has to be done on the text.
